# 🌙 Islamic Calendar 2026 — Hijri 1447/1448

A modern, mobile-friendly Islamic Calendar web app showing all 2026 dates with Gregorian-to-Hijri mapping, important Islamic events, and English/Urdu toggle.

## ✨ Features

- 📅 Full 2026 calendar with Hijri dates
- 🕌 Key events highlighted (Eid, Ramadan, Hajj, Laylatul Qadr)
- 🌐 English / Urdu toggle with RTL support
- 🌗 Dark & Light mode
- 📱 Mobile-first, swipe between months
- ⚡ Fast — built with React + Vite

## 🚀 Run Locally

```bash
npm install
npm run dev
```

## 🛠 Tech Stack

React · Vite · Tailwind CSS · Zustand · Framer Motion
